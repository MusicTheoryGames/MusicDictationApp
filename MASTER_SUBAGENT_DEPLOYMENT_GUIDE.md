## 🆘 **TROUBLESHOOTING**

### **Common Validation Failures:**

**Beat Count Errors:**
- Check rhythm math carefully
- Use beat maps: 4/4 (q=1, h=2, w=4), 6/8 (8=1, q=2, qd=3)
- Ensure all measures add up correctly

**Tonality Violations:**
- Stick to C major: C-D-E-F-G-A-B (any octave)
- A minor natural: A-B-C-D-E-F-G
- A minor harmonic: A-B-C-D-E-F-G# (raised 7th only)

**Similarity Issues:**
- Make changes smaller (1 note instead of 2)
- Keep same rhythm across options when possible
- Change notes by step (C→D) rather than leap (C→G)

**Classical Style Problems:**
- Add more stepwise motion
- Resolve leaps by step in opposite direction
- Use appropriate contour for phrase length
